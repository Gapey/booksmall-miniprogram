### 使用方法

1. 开发者服务器和数据库已部署在云服务器上（<http://119.29.28.23:8888>），无需再部署，不过还是提供了服务器代码和数据库脚本。

2. 数据库及服务器部署（可跳过）

    进入service文件夹
    
    生成数据库
    
    ```
    source mall.sql
    ```
    安装依赖包
    ```
    npm install
    ```
    启动服务
    ```
    node mall.js
    ```
    服务器使用8888端口

3. 请在https://mp.weixin.qq.com/ 得到appid和secret，微信开发者工具中设置appid。
在mine.js中设置好appid和secret。
   
4. 开始使用小程序
